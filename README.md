# SoulWrap™ — Existential Wellness Solution

A deadpan product landing page for a fictional existential wellness brand.

https://jzstafura.github.io/soulwrap/

## About

SoulWrap™ is a parody product for entertainment purposes. It's a minimalist, luxury-aesthetic landing page that satirizes wellness culture and infomercials while exploring the absurdist nature of existential dread.

## Features

- Single-page landing site with smooth scrolling
- Responsive design (mobile-friendly)
- Minimal, luxury wellness brand aesthetic
- Comparison table for existential treatments
- Product testimonials (fictional)
- CTA redirects to existential philosophy archive

## Development

No build tools required. Everything runs in vanilla HTML/CSS/JavaScript.

To run locally:
```bash
# Open index.html in any modern browser
# Or use a simple server:
python3 -m http.server 8000
# Then visit http://localhost:8000
```
